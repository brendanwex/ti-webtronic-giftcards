
## Introduction

Allows you to accept digital gift cards using Webtronics Gift Card Manager - More Info @  [Webtronic](https://webtronic.ie)

## Features

- Customized gift cards with your branding
- Gift Cards delivered automatically through email
- Accept the gift cards you sell anywhere, including in-store, through Webtronic mobile apps, or during checkout.
